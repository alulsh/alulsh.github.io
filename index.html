About Me

I taught myself to build websites at age 10, at the dawn of Web 2.0. I have been working professionally with web technologies since 2009, starting off as a SharePoint site collection administrator while an intern in college. For the past 3 years I have been working full time as a SharePoint professional, wearing both system administrator and developer hats.

I am an expert at applying the Defense Information Systems Agency (DISA) Security Technical Implementation Guides (STIGs) to SharePoint farms. I wrote a set of PowerShell functions that can STIG a complete SharePoint farm (SharePoint 2010, IIS 7, and SQL Server 2012) in less than 5 minutes. I can also write a Site Security Plan in less than a week.

Outside of SharePoint, I am currently trying to master client-side and server-side JavaScript. I currently serve as one of the leads for Women Who Code DC's Front End Hack Nights, helping local women learn HTML, CSS, and JavaScript.

Apart from technology I enjoy music, playing guitar, snowboarding, cooking, strength training, and running.
